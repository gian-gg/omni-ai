
# Omni Client (Mobile)

Expo + React Native mobile application for Omni.

This app is the primary user interface for logging transactions, todos, and notes using natural language, backed by the Omni FastAPI server.

---

## Tech Stack

- Expo (SDK 54)
- React Native
- TypeScript
- Expo Router
- Bun (package manager)
